🧬 University Research Project 2024/25

AI-Based Urine Microscopy
Analysis for Disease Detection

Leveraging deep learning and computer vision to automate urine sediment microscopy analysis, enabling faster, more accurate, and accessible disease screening.

🧪 Try Demo 📄 View Documents 📚 Explore Research
⚠️

The Problem

Traditional urine sediment analysis is manual, time-consuming, and prone to human error. It relies heavily on trained laboratory personnel, making it inconsistent and inaccessible in resource-limited settings.

💡

Our Solution

We propose an AI-powered system that uses convolutional neural networks (CNNs) and object detection models to automatically classify urine sediment particles, providing rapid and reliable diagnostic support.

Key Features

What Our System Offers

A comprehensive AI pipeline from image input to diagnostic insight.

🖼️

Image Preprocessing

Automated noise reduction, contrast enhancement, and normalization of microscopy images for optimal model input.

🧠

Deep Learning Classification

State-of-the-art CNN architectures trained to classify urine sediment particles across multiple diagnostic categories.

🎯

Object Detection

Real-time localization and identification of individual particles within microscopy slides using YOLO-based detection.

📊

Quantitative Analysis

Automated cell counting and concentration estimation to support clinicians in diagnostic decision-making.

📱

User-Friendly Interface

Clean web-based interface allowing lab technicians to upload images and receive results in seconds.

📋

Report Generation

Structured diagnostic reports with particle counts, classifications, and confidence scores for each analysis.
